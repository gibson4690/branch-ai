# AGENTS.md

This document describes the agentic architecture for question and answering business insights from a set of pandas dataframes.

## Agentic Architecture

### Graph
START -> DataConciergeAgent (creates an Analysis Plan, which is a list of insights to analyse)
DataConciergeAgent -> ReviewerAgent (routes to either DataEngineerAgent, DataAnalystAgent, or ExecutiveAgent)
ReviewerAgent -> DataEngineerAgent (Perform data queries and provides corresponding data dictionary of the queried data)
DataEngineerAgent -> ReviewerAgent (Back to ReviewerAgent to decides next step)
ReviewerAgent -> DataAnalystAgent (Uses the data to answer the selected insight, decides plots to show)
DataAnalystAgent -> ReviewerAgent (Back to ReviewerAgent to decides next step)
ReviewerAgent -> ExecutiveAgent (Creates an executive summary based on all completed insights)

### Schema

#### State
- business_question: the question asked by the user
- analysis_plan: analysis plan generated by the DataConciergeAgent
- executive_summary: executive summary prepared by ExecutiveAgent
- data_catalog: data_catalog loaded by get_data_catalog()
- follow up question: suggested follow up questions provided by ExecutiveAgent

#### Analysis Plan
List of Insights. Each Insight contains the following:
- insight_question: Question generated by DataConciergeAgent
- insight_short_answer: Short answer to the insight question, less than 15 words
- insight_long_answer: Longer answer to the insight question, less than 50 words
- insight_key_facts: data evidence to support the answers
- data_query_requests: list of required data queries needed
- insight_datasets: dict[str] dictionary of datasets.  
- insight_data_catalog: dict[str] data catalog of insight_datasets
- charts: dict[str]
- completed: boolean, True if agent is able to answer the question satisfactorily

#### datasets
Follows the following example structure
{
    "avg_wait_time_by_month_and_branch":
        {
            "dataset_name":"avg_wait_time_by_month_and_branch",
            "df": (pandas dataframe object)
        }
}

### Tools

query_data() : use parameters to perform data queries such as groupby, aggregation functions (count, mean, min, max, etc), sort, filter by values within a field

get_data_catalog() : retrieves the data catalog for all datasets (each dataset is a python dataframe, loaded from a data folder of csv files)
